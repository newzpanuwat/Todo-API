# Todo API

## API Endpoints

## Our API will expose the following RESTful endpoints.

### Endpoint	                        
* POST /signup	                    
* POST /auth/login	             
* GET /auth/logout	               
* GET /todos	                       
* POST /todos	                       
* GET /todos/:id	                 
* PUT /todos/:id	                   
* DELETE /todos/:id	                
* GET /todos/:id/items	          
* PUT /todos/:id/items	            
* DELETE /todos/:id/items
